# 数理小精灵：十之秘境 v2

> 当前版本：v0.39.0

一款针对5-7岁儿童的数学学习RPG游戏，通过宠物对战的方式练习10以内的加减法。

## v2 架构重构

基于 ES Modules 的模块化架构，从原版单文件重构为清晰的分层设计：

```
js/
├── main.js              # 入口
├── config/
│   ├── constants.js     # 游戏常量（状态、成长参数、难度权重等）
│   └── pokemon-data.js  # 宝可梦/道馆/天王/冠军数据
├── core/
│   ├── game.js          # 游戏主控制器（协调所有系统）
│   └── state.js         # 事件总线 + 状态存储
├── systems/
│   ├── battle.js        # 战斗系统（遭遇生成、伤害计算）
│   ├── math-engine.js   # 数学题引擎（加减法生成、选项生成）
│   └── pet.js           # 宠物系统（创建、升级、进化、药水）
└── ui/
    ├── renderer.js      # DOM渲染（所有UI更新集中管理）
    └── effects.js       # 视觉特效（攻击、星星、答题反馈）
```

## 游戏特色

- **宠物养成系统**：选择初始宠物，通过战斗获得经验值升级进化
- **回合制战斗**：正确回答数学题来攻击敌人
- **9大关卡**：8个道馆 + 四大天王 + 联盟冠军
- **难度递进**：从简单加法逐步引入减法和更大数字
- **儿童友好设计**：大按钮、明亮色彩、答题反馈动画

## 如何运行

直接用浏览器打开 `index.html` 文件，或访问 GitHub Pages。

## 技术实现

- 纯前端，无构建工具依赖
- ES Modules 模块化（`<script type="module">`）
- 事件驱动的状态管理
- 响应式设计，支持手机、iPad 和 PC
